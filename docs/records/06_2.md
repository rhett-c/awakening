### 音乐与天国

**零**

当安德拉斯·席夫步入音乐厅，空气忽然变细了。

公布曲目的一刻，我刚从他是否会重演因烤鸭取消演出的事故的疑虑中抽身。紧接着，一连串近乎拳击的惊喜将人击中——像一场私密而迟来的宗教经验。

也许恰逢复活节。

我看见银白的发丝在光中浮动，黑衣上细小的起球，台前花饰的微尘，以及舞台深处管风琴冷静的反光。

<br/>

**一**

沉入音乐久了，才慢慢辨认出某种秩序。

譬如线条。在歌唱中，它是呼吸，是一缕几乎不可见的生命延伸；在器乐中，它是时间本身被拉长后的痕迹。

当人类最初以声音赞美造物主时，是否也触及过这样的经验——一种肉身内部发生的启示？

于是人选择了音乐，各式各样的音乐。

但音乐仿佛是一种更高维度的存在。

<br/>

**二**

依稀记得西方古典音乐几乎是从神性中被“分离”出来的。

约翰·塞巴斯蒂安·巴赫的音乐仍然牢牢嵌在神学结构之内。哪怕脱离礼拜场景，也像一种声音的神学：秩序即救赎，对位即共同体。音与音之间彼此承担，如同信徒之间彼此代祷。

那首《哥德堡变奏曲》的咏叹调，旋律并不高声宣告，它更像是某种已经完成的世界——安静、封闭、完满。仿佛乌托邦不在未来，而在一段被不断回返的主题之中。

然而，历史并未停在这里。

到了约瑟夫·海顿与沃尔夫冈·阿马德乌斯·莫扎特，神性开始松动。音乐从教堂走向宫廷，再走向更广阔的公共空间。神不再是唯一的听众，人也逐渐成为目的本身。于是旋律变得透明、均衡、可理解。仿佛乌托邦被重新定义为一种可共享的美，一种理性可以抵达的和谐社会。但这种平衡终究脆弱。

<br/>

**三**

真正的断裂，或许发生在路德维希·范·贝多芬那里。

在他的音乐中，神不再是外在的对象，而被推入人的内部。斗争、挣扎、意志——这些本不属于宗教仪式的元素，开始占据中心。

《田园》看似回归自然，却已经不是伊甸园，而是经过主观意识重构的自然。它更像是一种“人类版本的创世”。

而到了晚期奏鸣曲——比如Op.109——事情变得更加复杂。音乐不再试图描绘世界，而是在自身内部生成一个世界。主题被撕裂、重组、超越，仿佛一场没有神学支撑的自我救赎。

这时我们或许不得不提问：当音乐脱缰于宗教，它是否开始取代宗教？

<br/>

**四**

而弗朗茨·舒伯特给出了一个更暧昧的答案。

他的音乐既不属于教堂，也不完全属于英雄式的人类精神。它更像是在两者之间徘徊——一种未完成的祈祷。

那些钢琴小品、即兴曲，短暂、脆弱，仿佛随时可能消散。旋律一再试图升起，却又在某种无形的重量下回落。

若说巴赫的音乐是已然实现的天国，那么舒伯特更接近一种失落的天国记忆。而这种记忆，本身就带着乌托邦的影子——它不存在，却不断被想起。

<br/>

**五**

回到这一晚。

当席夫弹奏这些作品时，我突然意识到一个微妙的转变。观众席中的人，安静、专注，几乎带着某种敬畏。这种敬畏并不指向某个明确的神祇，而指向音乐本身。

音乐厅在那一刻，像一座没有神像的教堂。

于是问题变得尖锐起来：人是否已经把音乐当成一种宗教？

我没有答案。也许音乐有自己的答案。

<br/>

音乐自己的答案来自，它始终处于一种张力之中。它既指向超越，又无法真正抵达；既生成共同体，又是私人；既像宗教，又不止于宗教。

真正的天国恐怕也从不落地。一旦落地，它就变成制度、规范，甚至权力。而音乐无法被彻底制度化，它只能在瞬间发生，像音乐厅的空气忽然变细，像某种无法证明、却被反复经验到的“临在”。

<br/>

**六**

所以问题或许不在于音乐是否成为了宗教。而在于，当神逐渐沉默，人是否需要一种新的方式，去安放天国的位置。

音乐恰好提供了一种持续悬置的状态。在这种状态里，乌托邦既被召唤，又被推迟；神性既被体验，又被怀疑。

而我们，只是在聚光灯外围静坐，等待下一次，线条再次出现。

&emsp;&emsp;阿西娅走在街道上，寒风灌进她的大衣，她不禁缩了缩脖子。街道上几乎没什么人，现在已经接近回家吃饭的时间。明明一个小时天还是亮的，一转眼的功夫镇子上的所有建筑就都染上了一抹黯淡的紫罗兰色。不过阿西娅还不想回家。

&emsp;&emsp;十二月的天气一如既往的糟糕，阿西娅还没走出几步，天上就飘起了大雪，她抬头望向漆黑的天空，雪花从视野不能及的高处缓缓飘落，就像从魔术师的帽子中凭空变出来的白鸽一样。仅仅一面栅栏外，荒凉的冰原笼罩在夜色中，一条灰色的石子路从镇子的出口笔直延伸向远方，石子路的边界在一望无际的白茫茫的雪地中不甚清晰。尽管在这里还看不真切，但阿西娅知道在道路尽头有一座四四方方的黑色建筑孤零零的矗立在那儿。

<br/>

&emsp;&emsp;阿西娅自从两年前开始就经常梦见一座雪白的城市。城市笼罩在一种柔和的鹅黄色光晕中。那里的房屋像是大大小小的齿轮拼接而成，那里的孩子披着丝绸质地的轻盈洁白的纱衣，在齿轮与连杆间嬉笑玩耍，或是坐在巨大的转轮上静静地听大人们讲故事。在梦中，阿西娅也是他们中的一员，在那里她有很多朋友，她和所有人相处得都很自在，不必担心遭受其它孩子捉弄，也不用为了附和他们的笑话而殚精竭虑。她有时会沿着一个方向一直走，直到走到城市的尽头，一路上她会看见许多瑰丽的奇景。大多数梦中的场景醒来后都会遗忘，但偶尔也有给阿西娅留下深刻印象的景色。有一次她来到一座耸立的螺旋形高塔脚下，无数棱角分明的巨大钢制结构沿着高塔的斜面首尾相连，直指天空。从那座塔顶传来无线电广播嘈杂的声音。她循着声音的方向开始攀登高塔，道路螺旋上升，好似看不到尽头。阿西娅只觉得自己走了好久好久，周围刮起凛冽的风，下方的景象都已经变得微小而模糊不清了，自己仿佛站在云端俯瞰整片大地。头顶的无线电广播愈加清晰了，阿西娅听不懂广播里说的内容，但是她觉得广播里正在说的是与她、与这个国家的每一个人都息息相关的事情，她产生了一种由衷的自豪感。她又往上走了一段路，直到一扇门出现在自己面前。她知道自己该回去了。

&emsp;&emsp;这样的梦通常会一连持续很多天，直到一个声音温柔地告诉她“该离开了”，她才会依依不舍地和朋友们道别，然后穿过一道狭窄逼仄的门，回到清醒的世界。不过阿西娅的梦在一周前就中断了。那座美丽的城市彻底从阿西娅的梦中消失了，在那之后，她就只能梦见一些模糊不清的无意义的画面。这使她感到分外低落，一连几天的课上都无法集中精神。她望着窗外的冰原，思索着那座城市会在哪里，自己又该如何回到那里去。期间，班上那些孩子有找过几次麻烦，但阿西娅没有理睬他们所说的话，她甚至没有意识到他们走到了自己面前。也许这之后会给她带来更大的麻烦，但是如前所述，她再也无法将注意力集中到眼前这个阴沉暗淡的世界了。

<br/>

&emsp;&emsp;阿西娅艰难地跨过雪地，来到了那栋四四方方的黑色建筑前。她不由得为能再次见到那座美丽的城市而欢欣鼓舞。即使自己恐怕再也没办法回到那里，但只是在远处看两眼，都能给她沉闷的生活带去一些希望。她小心翼翼地从研究所未上锁的窗户处翻进去，驾轻就熟地穿过曲折迂回的长廊。空荡荡的研究所里回荡着仿佛永恒不变的滴答声和泵气声，从窗户往里看，可以看见每一间房间里都摆满了叫不出名字的仪表和机器。阿西娅轻轻推开走廊尽头的一扇门。在这间房间中央摆放着一面镜子。镜子中倒映着一座雪白的城市。

## 3 <div class="covered-text">夜鹭豌豆</div>

伊芙七岁时见到了白色的火焰。那天她父亲驾车穿过17号公路，迎面撞上了一头鹿。伊芙的耳鸣渐渐消退，她睁开眼睛，发现鹿的尸体注视着自己。车撞开了公路护栏，将鹿牢牢地卡在前引擎盖上，然后开始燃烧。伊芙闻到了焦糊而恶心的气味，明亮的橘黄色的火焰在她面前跳动着，令人眩晕。

父亲把伊芙抱出车子，然后去抢救行李。伊芙坐在地上，把手指戳进泥土里。泥土是冰凉的。这时候，她看见火焰变成了白色。伊芙小时候喜欢盯着太阳看，然后闭上眼睛，视野里就会出现一团模糊的白色。火焰就是那种颜色。伊芙闭上眼睛，白色的光晕收拢起来，变成了鹿的形状。

鹿的幽灵说：“冬天。”

伊芙的父亲说：“伊芙，别怕。我们可以离开了。”

伊芙握住父亲的手。拖车的橙色灯光一闪一闪，火焰已经不见了。父亲的车的凹痕里只有漆黑的颜色。

据说很久以前，克莱温什有一位女巫。这个女巫可以唤回亡灵。人们点燃火堆，一天过后，那里只剩下苍白的灰尘。一些故事声称女巫的灵魂就困在这些灰尘里，起风时便会游荡和哭泣。在另一种变体中，火焰成为了女巫召唤自己的仪式，让灵魂在死后回归。

伊芙相信某个祖先使她能够听见灵魂。燃烧是一种诱导，引出了身体里尚未离开的部分——尽管只能讲述一两个无关紧要的词语。很长一段时间里，她对此既没有感到兴奋也没有感到恐惧。伊芙在克莱温什找到了工作，生活下来，像一切这个城市中的人一样。

一天早上，伊芙在电话里得知了父亲的死讯。她的父亲如今将会躺在某处，面色平静，被陌生人和亲人环绕。最终他会被埋葬在土里，或者被火化，变成容器里的灰烬。火，伊芙想起，女巫的亵渎的火焰会唤醒她沉睡的父亲。她会唤醒她的父亲。

据说伊芙的父亲没来得及讲述遗言。据说伊芙后来没有到场。“那个人是伊芙。”她听见有人说道。她站在停车位旁、站在镜子前、站在院子里，然后一瞬的思绪冲刷过她全身。伊芙转过头，没有人注视着她。

伊芙买来了火柴。只有一指长，每盒二十支。她在午夜时点燃树叶，聆听噼啪作响的火焰。后来伊芙买来蜡烛，然后盯着摇曳的微弱火光。她闭上眼睛。火焰就是那种颜色。金色，橙色，红色。伊芙看着自己的手，她的手是充满血的红色。这只手曾经会因为被紧握而疼痛。

在某一个夜晚，伊芙离开了家。月光诱发了某种潜伏的疯狂。她发现自己来到了一片无人造访的林地。月亮就在火堆的正上方，火焰越来越高，像一颗飘摇的红色树苗。伊芙解开布包，看见了一把银色的小刀。

她的小指是红色的，如同一支火柴。火光闪烁，被林风吹向高处。黑色的树叶从天上掉落下来，伊芙深深地呼吸，然后看见了白色的火焰。白色的幽灵。

她自己的灵魂说道：“一颗心脏。”

它开始哭泣。哭声平静而悲伤。一个巨大而压抑之物在伊芙的胸膛中颤抖。

“我很害怕。”伊芙说，“我很抱歉。”

火堆噼啪作响，仿佛火焰还尚未熄灭。面前只有一片黑色的林地，泥土是冰凉的。在这一刻，伊芙终于可以相信，她是那么深爱、那么怀念着她的父亲。

月亮落下，此后伊芙再也没有看见过白色的火焰。

### 标题:戴着斑鸠烘焙帽的烘焙师斑鸠

……文字工厂穆夫提脱脱不花雪亮漆黑的平方根指着我说出苝肉豆蔻苝流水线苝伏地魔苝热干面我的工作就是用刮刀把亘古不停涌动的文字奶油切割成一个一个文字方块一块接着一块接着一块接着一块接着哦不我不能让思维跟着传送带运行而且那是同事至于我的手他操纵装有整部穆赫兰道的电钻把文字草莓打成黏糊糊的文字果酱为什么人们总说起给思想的食物那给思想的饮用水给思想的庇护所和空调又如何呢思想会嫉妒地从薰衣草色美甲中挤出二十二个香蕉字母涂抹在大腿上与铜质反应堆调情以对外国语发动核打击吗哦不长话短说显然被我吃掉的粉巧克力和香槟柠檬冰激凌球的焦虑感传染给了穆夫提他假装思考了一会温柔如春含着嫩槟郎般劝我脱下防接触服离开文字工厂直到天黑前不要回来这是什么意思天上的三个月亮照耀无有阴影我没有通向庇护所的八十阶楼梯时间太多喷泉上的鸽子踱来踱去米粮川石榴用来贿赂他们啊是有美丽脖子的龙在讨论谁去周日请那个不受欢迎的家伙捉蚯蚓我感到很吃惊一半因为这是霸凌另一半因为尖尖的喙努力撕开了空气或是磁场或是虚空中成对出现又彼此湮灭的社会规范露出一个洞来排练皮影戏而我刚刚为了一双闻起来像芫荽的鞋把影子典当掉了为能谋得一官半职决然钻进洞里滑溜溜却很窄转身再看旧肉体留在了门外尾羽流转从未见过的颜色原来颜色远比躲在地狱的化学家想象的主观得紧可惜我前后碰了好几块松根也没能学会飞行撞得我不必匮乏歇在平静水边水里有一个黑色的正方形多么完美的正方形与白边能看见门后的干净明亮的吧台摆着疏于安保的食物和酒我抬起鳞片厚重的脚爪尖试探那水他真是冰凉降落到吧台上是久违的十娃子我竟然曾忘了他正是我那不知出处的紧迫感撒着品红色酸味花瓣大概还有几百片这时候读者也该知道了那种喻示这几百片花瓣和千千万万红色团花的花瓣有一种共同的形式以二十二个字母为界我看见圣书里记载的世界起源宇宙是海洋玻璃纸汤塑料中无限延伸的文字只有二百个仍悬而未决念出这咒语放黑夜进入我不再需要食物净水庇护所我让所有沙变成摄像机的蓝宝石磨片做最后的点缀但十娃子呼喊我的名字叫红我的喙是电钻带起头脑中的整部鲁赫那玛做我早已烂熟的工作把一块一块精美的宇宙蛋糕重新搅拌直至幼滑如初文字快乐地回到虚空的梳妆台口袋里而后再一次再一次再一次再一次……
